# Экран загрузки (Loading Screen)

Простой и современный экран загрузки, реализованный с использованием чистого стека технологий: **HTML5, CSS3 и JavaScript**.

## 🚀 Технологии

- **HTML5**: Структура страницы.
- **CSS3**: Стилизация, CSS переменные для легкой настройки цветов и ключевые кадры (`@keyframes`) для анимаций.
- **JavaScript (Vanilla JS)**: Логика управления прогрессом и обновление интерфейса без использования сторонних библиотек.

## 🛠️ Как запустить

Для запуска проекта не требуется установка зависимостей или сборка:

1. Клонируйте репозиторий или просто скачайте папку `loading-screen`.
2. Откройте файл `index.html` в любом современном браузере.

## 📂 Структура проекта

- `index.html` — основной HTML-файл.
- `style.css` — все стили и анимации.
- `script.js` — логика симуляции загрузки.
- `README.md` — описание проекта.
